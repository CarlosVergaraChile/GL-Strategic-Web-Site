## Requisitos

- **Formato:** JPG o PNG
- **Tamaño recomendado:** 300x400px (aspect ratio 3:4 para flip cards)
- **Compresión:** Optimizadas para web
- **Resolución:** Mínimo 150x200px, máximo 500x667px

## Notas Importantes

1. **Gastones:** No confundir las fotos
   - Gastón L'Huillier T. (Socio Principal) = FOTO A COLORES
   - Gastón L'Huillier Ch. (Consultor) = FOTO EN B/N

2. **Ubicación:** Todas las fotos van en la carpeta `team/`

3. **Nombres de archivo:** Seguir el patrón `NN-apellido-nombre.jpg` para mantener consistencia

4. **Orden:** Respetar el orden numérico (01-25) según la lista anterior

## Cómo agregar imágenes

1. Descargar fotos de la carpeta de Google Drive: `GL Strategic > Fotos Equipo`
2. Renombrar según el patrón especificado
3. Comprimir si es necesario (max 100KB por imagen)
4. Subir a GitHub usando la interfaz web o Git CLI
5. Actualizar `index.html` para mostrar las imágenes

## Logo

- **Archivo:** `logo.png` o `logo.svg`
- **Ubicación:** Se mostrará en la barra de navegación (navbar)
- **Tamaño:** Aproximadamente 40x40px en el navbar

**Fuente de archivo:** Google Drive > GL Strategic > Logos/
